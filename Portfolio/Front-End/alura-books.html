About the project

Alura Books is a fictitious e-commerce developed during an Alura course with the aim of simulating a virtual store that sells books about technology. The project was built entirely with HTML and CSS, while JavaScript was used to implement the image slider, offering an interactive experience to users.

In the Alura Books project, I applied my knowledge of HTML, CSS, and JavaScript to create an engaging shopping experience for users interested in technology books. Using HTML, I structured the elements of the page, from the header to the footer, ensuring a logical and intuitive organization of content. With CSS, I styled the website layout to provide an attractive and cohesive visual experience. Additionally, with JavaScript, I implemented the functionality of the image slider, offering a dynamic and interactive way to view the products available in the store. This project demonstrates my ability to develop functional and visually appealing web interfaces, integrating basic web technologies to create an enhanced user experience.
