About the project

Alura Fone is a digital keyboard project developed with the goal of learning to manipulate the DOM using JavaScript. The keyboard is designed to mimic a phone keyboard, allowing users to input numbers and perform basic operations.

In the Alura Fone project, I explored DOM manipulation in JavaScript to create a functional digital keyboard. Using JavaScript, I implemented the keyboard logic, including capturing click events on the numeric buttons and displaying the entered numbers. Additionally, I worked with HTML and CSS to structure the layout of the keyboard and style it according to the proposed design. This project was a valuable opportunity to enhance my skills in JavaScript and become familiar with dynamic DOM manipulation, essential for the development of interactive web applications.
