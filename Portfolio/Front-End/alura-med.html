About the project

Alura Med is a web application that serves as an electronic agenda for healthcare professionals, allowing them to schedule appointments with patients and communicate through a direct chat. The application was developed during the Alura #7daysOfCode Challenge, primarily using HTML and CSS, with JavaScript applied for specific functionalities such as the appointment addition button.

The development of Alura Med during the Alura #7daysOfCode Challenge was an exciting opportunity to apply web development concepts and work with fundamental technologies such as HTML, CSS, and JavaScript. My contribution to the project included the implementation of interactive and responsive interface elements, ensuring an intuitive and efficient user experience. Through the creative and effective use of these technologies, we were able to create a functional and useful application for healthcare professionals, helping them manage their appointments more effectively and provide better patient care.
