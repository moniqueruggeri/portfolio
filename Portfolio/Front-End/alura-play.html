About the project

Alura Play is a fictitious website that mimics the layout of YouTube but exclusively displays videos from Alura. This project was developed entirely in HTML and CSS, with the aim of replicating the browsing experience of YouTube, offering a selection of educational videos from Alura.

In the Alura Play project, I used my skills in HTML and CSS to create a platform for Alura's educational videos with a similar appearance to YouTube. Using HTML, I structured the website's content, including video lists and navigation elements. With CSS, I styled the website's layout to mimic the design of YouTube, ensuring a familiar and intuitive experience for users. This project demonstrates my ability to develop attractive and functional web interfaces using basic web technologies.
