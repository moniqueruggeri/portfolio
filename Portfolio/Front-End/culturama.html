About the project

Culturama is a fictitious website that gathers the main free events in São Paulo. This project was developed during the "CSS: building layouts with grid" course, with the aim of applying responsive layout concepts using CSS Grid.

In the Culturama project, I applied my knowledge of CSS Grid to create a modern and functional layout that highlights free events in São Paulo. Using HTML to structure the content and CSS Grid to organize the elements of the page, I was able to create an intuitive and attractive user experience. The responsive layout ensures that the site is accessible on mobile devices, providing a consistent experience for all users. This project demonstrates my ability to use web technologies to create efficient and visually appealing interfaces.
