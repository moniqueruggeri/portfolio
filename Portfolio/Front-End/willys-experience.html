About the project

Willy's Chocolate Experience was a fraudulent event that took place in Glasgow, Scotland, where hundreds of people were deceived by a promised immersive experience inspired by the classic "Charlie and the Chocolate Factory". However, upon arrival, they found a warehouse in poor conditions. Faced with this incident, I undertook the project to redesign the Willy's Chocolate Experience website, aiming to improve crucial aspects such as spacing, contrast, and font, using Figma.

Before

After

In the redesign project of Willy's Chocolate Experience website, I utilized my skills in web design and development to create a more attractive and functional interface. Through Figma, I redesigned the website layout, ensuring adequate spacing between elements, improved contrast for readability, and a more legible font choice. These visual enhancements helped convey a more professional and trustworthy image to the website visitors, reinforcing the brand's credibility after the incident. All development was carried out using only HTML and CSS, demonstrating the effectiveness of these technologies in creating impactful and accessible web interfaces.
